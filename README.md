# DELTA — тестовое задание на соискание вакансии Frontend-разработчик

[Простое одностраничное приложение](https://nagraiv.github.io/delta-test/) на **React + TypeScript + Vite** для отображения аналитической таблицы показателей.

Функциональность проекта:

- Загрузка данных по fetch (имитация API)
- Автоматическое вычисление сумм по дочерним строкам
- Подсветка положительной/отрицательной/нейтральной динамики
- Недельные графики на Highcharts, раскрываемые по клику на строку
- Возможность изменения масштаба таблицы
- Препроцессоры SASS

---

## Стек технологий

- **React 19**
- **TypeScript**
- **Vite 7**
- **SCSS**
- **Highcharts**
